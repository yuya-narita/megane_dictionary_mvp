# メガネ辞書 MVP v2

この版は `index.html` をそのままブラウザで開いて動くように、データを `app.js` に埋め込んであります。

## 使い方
1. ZIPを解凍
2. `index.html` をブラウザで開く

## 操作
- 左右スワイプ：単語を切り替え
- 上下スワイプ：メガネを切り替え
- 長押し：メガネ一覧
- PCでは矢印キーでも操作可能

## 注意
スマホで開く場合は、フォルダごと展開してから `index.html` を開いてください。

- 縦スワイプ時は文字が縦だけ追従
- 斜めブレを抑えて読みやすさを改善

- v9のレイアウト崩れを修正

- カードモードは左右スワイプでカード切替
- バグカード5枚収録


## v12変更点
- カードモード操作を変更

  - 左右スワイプ：表裏フリップ
- カード裏面画像を追加
- カード切替時は表面に戻る


## v13変更点
- カードモード開始時は裏面表示
- 裏面のまま上下スワイプすると、表を隠したままカードをシャッフル
- 左右スワイプで表裏フリップ
- 裏面中はタイトルを隠して「？？？」表示


## v14変更点
- カードは基本的に常に裏面表示
- 上下スワイプ：裏面のままカード切替
- 左右スワイプ：表をチラ見せ、指を離すと裏に戻る
- カードモード開始時も裏面


## v15変更点
- 左右スワイプで表裏を切り替え
- 表にしたカードは勝手に裏へ戻らない
- もう一度左右スワイプで裏へ戻る
- 上下スワイプで次カードへ移動した時は裏面に戻る


## v16変更点
- 上下スワイプ時にシャッフル演出を追加
- 裏面のままカードが軽く揺れて入れ替わる
- 表面は見せず、カードを引き直している感覚を強化


## v17変更点
- 上下スワイプ時にデッキ演出を追加
- 裏面カードが複数枚重なり、配っているように揺れる
- シャッフル感をv16より強化


## v18変更点
- 上スワイプ：前面カードが上へ流れて、下のカードは固定
- 下スワイプ：前面カードが下へ流れて、下のカードは固定
- シャッフルより「カードを配る」感を強化
- ダブルタップでカードを全画面表示
- 全画面はタップで閉じる


## v19変更点
- シャッフル演出を安全化
- 本体カードは動かさず、演出専用の裏面カードだけを上下に流す
- 流れる途中で表面が見えてしまう問題を修正
- 表裏操作後でもシャッフル演出が消えないように修正


## v20変更点
- 全画面表示でも通常カードと同じ状態を維持
- 全画面で左右スワイプすると表裏切替
- 全画面で上下スワイプすると裏面のままカード切替
- 全画面でも裏面カードが上下に流れる演出を追加
- 全画面はタップで閉じる


## v22変更点
- マンガモードを追加
- 初期サンプル「その沈黙」5ページ入り
- マンガモード操作
  - 左右スワイプ：ページ送り
  - 上スワイプ：カードモードへ戻る
- 画像なしでも動くテキストマンガ形式
- 後で `manga.json` と画像パスを使って画像マンガ化可能
