# メガネ辞書 MVP v2

この版は `index.html` をそのままブラウザで開いて動くように、データを `app.js` に埋め込んであります。

## 使い方
1. ZIPを解凍
2. `index.html` をブラウザで開く

## 操作
- 左右スワイプ：単語を切り替え
- 上下スワイプ：メガネを切り替え
- 長押し：メガネ一覧
- PCでは矢印キーでも操作可能

## 注意
スマホで開く場合は、フォルダごと展開してから `index.html` を開いてください。
